HTML Headings

HTML headings are defined with the <h1> to <h6>

<h1> defines the most important heading, <h6> defines the least important heading

Headings are important

Search engines use headings to index the structure and content of web pages

Bigger Headings

Each HTML heading has a default size. However you can specify the size for any heading with the style attribue using the CSS font-size property
